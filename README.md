### XXE Data Access (⭐⭐⭐)

![image](https://github.com/user-attachments/assets/fd6a1e47-1bbc-471b-89d0-726d382abee3)

-> XXE Injection을 수행하는 XML 파일을 제작해 서버에 전송 (윈도우 환경이기 때문에 /system.ini 에 시스템 파일 존재)
